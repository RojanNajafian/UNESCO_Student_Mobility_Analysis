# UNESCO Tertiary Student Mobility Analysis

## Project Overview

This project analyzes international student mobility using UNESCO Institute for Statistics (UIS) data.

The dataset includes outbound mobility ratios of tertiary students across countries and years.

The analysis explores patterns, trends, outliers, and relationships between countries using data analysis and visualization techniques.

---

## Objectives

* Analyze global student mobility patterns
* Identify countries with highest and lowest mobility ratios
* Perform statistical analysis
* Detect outliers using IQR method
* Analyze correlations between countries
* Visualize trends using multiple plots
* Create pivot tables and grouped analysis

---

## Dataset Source

UNESCO Institute for Statistics (UIS)

Indicator used:

Outbound mobility ratio, both sexes (%)
